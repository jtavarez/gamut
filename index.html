Gamut finds the range of numbers in element provided, along with an array of 2 or more colors, and does the task of associating each point in the range to the color in the spectrum (or gamut. Get it?). It's easy enough to use with tables or other elements where you want to associate data with color.

Download Version 0.7 or view on Github

Getting Started

  1. Download the js file and link to it from the web page
  2. Call the gamut() function. This requires 2 parameters to get started:
$('table').gamut({
  item : 'td',
  colorArray: ['ff0000', '000000']
});
